Lombok

JSR 269 API 规范

Lombok实战

  • @Data [包含了@Setter @Getter @ToString @EqualsAndHashCode]
  • @Getter
  • @Getter(AccessLevel.PROTECTED)
  • @Setter
  • @Setter(AccessLevel.PROTECTED)
  • @NoArgsConstructor
  • @AllArgsConstructor
  • @ToString
  • @ToString(exclude=“column”)
  • @ToString(exclude={“column1”,”column2”})
  • @ToString(of=“column”)
  • @ToString(of={“column1”,”column2”})
  • @EqualsAndHashCode (同@ToString)
  • @EqualsAndHashCode(exclude=“column”)
  • @Slf4j [适用于logbook日志框架]
  • @Log4j [适用于log4j日志框架]

Lombok实际使用需注意的点

  • 在类需要序列化、反序列化时详细控制字段时，例如：Jackson json序列化
  • 使用Lombok虽然能省去手动创建setter和getter方法等，但是却降低了源码文件的可读性的完整性，以及阅读源码的舒适度
  • 使用@Slf4j还是@Log4j，看项目使用的日志框架，教程中使用Logback，所以使用@Slf4j
  • 选择合适的地方使用Lombok，例如pojo是一个好地方
